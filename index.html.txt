        <section id="math-foundations" class="section">
            <h2>Mathematical Foundations</h2>
            <p>This section provides the complete mathematical formalization of the Capability-Based Coordination framework, including formal definitions, proofs, and computational specifications.</p>

            <h3>Core Mathematical Definitions</h3>

            <h4>1. Personal Reality Framework (PRF) Complete Specification</h4>
            <div class="math-formula">
                PRF(A,t) = ⟨BROA+(t), C(t), ATCF(t), XSC(t), FOP(t)⟩
            </div>

            <p>Where each component is formally defined as:</p>
            <ul>
                <li><strong>B(t):</strong> Belief system with probability distributions over propositions</li>
                <li><strong>R(t):</strong> Rule system governing decision-making and action</li>
                <li><strong>O(t):</strong> Ontological categories structuring experience</li>
                <li><strong>A+(t):</strong> Authenticity criteria for identity preservation</li>
                <li><strong>C(t):</strong> Capability specifications for coordination</li>
                <li><strong>ATCF(t):</strong> Adaptive Temporal Coherence Function</li>
                <li><strong>XSC(t):</strong> Cross-scale correspondence patterns</li>
                <li><strong>FOP(t):</strong> Future-oriented projections</li>
            </ul>

            <h4>2. Adaptive Temporal Coherence Function (ATCF)</h4>
            <div class="math-formula">
                ATCF(A, t₁, t₂) = α·HC + β·PI + γ·PC + δ·MCC
            </div>
            <p>With constraints: α + β + γ + δ = 1, all weights ∈ [0,1]</p>

            <p>Component definitions:</p>
            <ul>
                <li><strong>Historical Continuity (HC):</strong> Maintains narrative connections to past experience</li>
                <li><strong>Present Integration (PI):</strong> Coordinates current beliefs, emotions, goals, and actions</li>
                <li><strong>Prospective Coherence (PC):</strong> Aligns future plans with core identity</li>
                <li><strong>Meta-Constructor Capacity (MCC):</strong> Ability to modify beliefs while preserving identity</li>
            </ul>

            <h4>3. Capability-Based Propositions</h4>
            <p>Replace abstract truth-bearers with functional ability specifications:</p>
            <ul>
                <li><strong>Generate:</strong> Minimal abilities to produce relevant content</li>
                <li><strong>Maintain:</strong> Abilities to preserve relationships across contexts</li>
                <li><strong>Coordinate:</strong> Abilities to achieve alignment with other agents</li>
            </ul>

            <h4>4. Modal Logic Extensions</h4>
            <div class="logic-box">
                <h4>Enhanced Logical Systems</h4>
                <p><strong>Enhanced ATL:</strong> ⟨⟨A⟩⟩ᶜφ means "Coalition A has capability-based strategy to achieve φ while preserving authenticity"</p>
                <p><strong>Enhanced STIT:</strong> [i stit: φ]ᵗᶜ means "Agent i sees to it that φ while maintaining temporal coherence"</p>
            </div>

            <h4>5. Bootstrap Resolution Proof</h4>
            <div class="highlight-box">
                <h4>Theorem</h4>
                <p>Any agent capable of questioning normative authority already demonstrates the ATCF requirements for ethical coordination.</p>
                <p><strong>Proof sketch:</strong> The act of asking "Why should I be ethical?" presupposes historical continuity, present integration, prospective coherence, and meta-constructor capacity—the very components that ground normative coordination.</p>
            </div>

        // Add active state to navigation
        const sections = document.querySelectorAll('.section');
        const navLinks = document.querySelectorAll('.nav-link');

        function updateActiveNav() {
            const scrollPos = window.scrollY + 100;
            
            sections.forEach((section, index) => {
                const sectionTop = section.offsetTop;
                const sectionHeight = section.offsetHeight;
                
                if (scrollPos >= sectionTop && scrollPos < sectionTop + sectionHeight) {
                    navLinks.forEach(link => link.classList.remove('active'));
                    if (navLinks[index]) {
                        navLinks[index].classList.add('active');
                    }
                }
            });
        }

        // Add CSS for active state
        const style = document.createElement('style');
        style.textContent = `
            .nav-link.active {
                background-color: var(--primary-blue);
                color: var(--white);
            }
        `;
        document.head.appendChild(style);

        window.addEventListener('scroll', updateActiveNav);
        window.addEventListener('load', updateActiveNav);
